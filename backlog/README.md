# Backlog — Silver Sage Publishing Website

Ideas, improvements, and session prep notes for the Silver Sage Books website.

Drop files here between sessions. Each file = one idea or task proposal. Use any format, but a good default:

```
# [Short title]
**Source:** [where the idea came from]
**Priority:** [high / medium / low]
**Description:** [what and why]
**Acceptance criteria:** [how we know it's done]
```

These items will be reviewed at the start of each Claude Code session and incorporated into the session prompt as appropriate.
